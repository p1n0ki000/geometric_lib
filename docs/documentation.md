# Project Documentation: geometric_lib

## General Description
Geometric Library is a library for calculating the areas and perimeters of a circle and a square.
The library can be used for educational purposes or later for implementing complex calculations. Each function includes comments.

## Function Descriptions

### Circle

**File**: `circle.py`

#### `area(r)`
Calculates the area of a circle given its radius.
**Formula**
S = π * r²

**Example**
area(2) # 12.56637061

#### `perimeter(r)`
Calculates the perimeter (circumference) of a circle given its radius.
**Formula**
P = 2 * π * r

**Example**
perimeter(4) # 25.13274123

### Square

**File**: `square.py`

#### `area(a)`
Calculates the area of a square.
**Formula**
S = a²

**Example**
area(2) # 4

#### `perimeter(a)`
Calculates the perimeter of a square.
**Formula**
P = 4 * a

**Example**
perimeter(2) # 8
